# Customer Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Business Intelligence tools. It provides insights into customer patterns, preferences, and spending habits to support data-driven decision-making.

## Project Structure

```
.
├── Analysis/
│   ├── Customer_Behavior_Analysis.ipynb    # Interactive Jupyter notebook with analysis and visualizations
│   └── Customer_Behavior_Analysis.sql      # SQL queries for data extraction and transformation
├── Data/
│   └── customer_shopping_behavior.csv      # Raw customer behavior dataset
└── README.md                               # This file
```

## Files Description

### Analysis Directory

**Customer_Behavior_Analysis.ipynb**
- Interactive Jupyter notebook containing:
  - Data loading and exploration
  - Statistical analysis
  - Data visualizations
  - Customer segmentation and insights
  - Findings and recommendations

**Customer_Behavior_Analysis.sql**
- SQL queries for:
  - Data cleaning and preparation
  - Aggregation and summarization
  - Customer metrics calculation
  - Database queries (if using a relational database)

### Data Directory

**customer_shopping_behavior.csv**
- Raw dataset containing customer shopping behavior records
- Includes customer demographics, purchase history, and behavior patterns

## Getting Started

### Prerequisites
- Python 3.x with Jupyter Notebook
- SQL database (if applicable)
- Required Python libraries:
  - pandas
  - numpy
  - matplotlib
  - seaborn
  - scikit-learn (if using machine learning)

### Usage

1. **Data Exploration**
   - Open `Analysis/Customer_Behavior_Analysis.ipynb` in Jupyter Notebook
   - Run cells sequentially to explore the data

2. **SQL Analysis**
   - Use queries from `Analysis/Customer_Behavior_Analysis.sql` in your SQL environment
   - Modify queries as needed for specific analysis

3. **Visualization & Insights**
   - Review visualizations and insights generated in the notebook
   - Use findings to inform business decisions
